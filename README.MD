## controls
*Mouse locks when you click on the window, press ESCAPE to unlock the mouse.*

* W,A,S,D / Arrow Keys = Move
* L-SHIFT / R-CTRL = Sprint
* Left Click = Shoot
* Right Click = Zoom
* Mouse Scroll = Change Weapon
